This python program will write down every text in paste.txt which will be copied into your device

follow these rules to run the .exe


1.Run the .exe and wait for antivirus scan if there are any
2.then try to copy any text 
3.First time allow antivirus to change paste.txt after that close the .exe and run it again

(recommended::always run the .exe with startup)

____________________how to start the .exe with startup______________________
|     1.First download or git clone the repository in your C drive         |
|     2.Make a short cut of the .exe and copy it                           |
|     3.Search startup in C drive                                          |
|     4.Or cd c:\ProgramData\Microsoft\Windows\Start Menu\Programs\startup |
|     5.Paste the short cut there                                          |
|          (The .exe will start with startup for every user)               |
